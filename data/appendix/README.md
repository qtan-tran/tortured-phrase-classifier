# Data Appendix

This directory contains data files for the *Knowledge Organization* paper appendices.

## Appendix A — Signal Registry Export (`appendix_a_registry_export.csv`)
All registry entries with warrant status, domain, prevalence, and performance metrics.
Generated by: `tpc registry-stats --candidates > data/appendix/appendix_a_registry_export.csv`

## Appendix B — Corpus Statistics (`appendix_b_corpus_statistics.csv`)
Corpus composition for evaluation: positive (retracted), negative (clean), synthetic.
Generated by: running `notebooks/01_data_acquisition.ipynb`

## Appendix C — Full Ablation Results (`appendix_c_ablation_results.csv`)
All 7 layer combinations × 4 corpora × 5 metrics.
Generated by: running `notebooks/05_combined_evaluation.ipynb`

## Appendix D — Novel Phrase Candidates (`appendix_d_novel_candidates.csv`)
Top-50 novel candidate phrases detected by Layer 3 in 2024–25 corpus.
Columns: span, log_perplexity, proposed_canonical, domain, expert_assessment, recommended_status.
Generated by: running `notebooks/04_layer3_mlm_perplexity.ipynb`

## Appendix E — Registry Growth Timeline (`appendix_e_registry_growth.csv`)
Monthly data: registry size vs. cumulative paper mill retractions.
Columns: year_month, cumulative_confirmed_signals, cumulative_retractions, pearson_r_lag0, pearson_r_lag3.
Generated by: `from tpc.evaluation.registry_growth import plot_registry_vs_retractions, export_appendix_e`

## Reproducibility
All appendices are fully reproducible from the code and public data sources.
Run `notebooks/06_figures_for_paper.ipynb` to regenerate all figures and appendix files.

Data sources:
- PubMed E-utilities API (public, no key required for ≤3 req/sec)
- Retraction Watch database (public CSV, see https://retractionwatch.com)
- Cabanac tortured phrase list (public, github.com/gcabanac/tortured-phrases)
- OpenAlex API (public, no key required)
