# Ground Truth Corpora

This directory stores evaluation corpora for the TPC benchmark.

## Files

### confirmed_tortured.tsv
Positive evaluation examples: (pmid, abstract, label="retracted") pairs confirmed
to contain tortured phrases. Generated by `tpc/acquisition/pubmed.py`.

### confirmed_clean.tsv
Negative evaluation examples: (pmid, abstract, label="clean") pairs from
non-retracted papers, domain-matched to the positive corpus.

## Generating the Corpora

```bash
# Requires: pip install -e .
# Optional: set NCBI_API_KEY environment variable for higher rate limits

python -c "
from tpc.acquisition.pubmed import fetch_retracted_abstracts, fetch_clean_abstracts
import pandas as pd

positive = fetch_retracted_abstracts(max_results=5000)
negative = fetch_clean_abstracts(max_results=5000)

pd.DataFrame(positive).to_csv('data/ground_truth/confirmed_tortured.tsv', sep='\t', index=False)
pd.DataFrame(negative).to_csv('data/ground_truth/confirmed_clean.tsv', sep='\t', index=False)
print(f'Positive: {len(positive)} | Negative: {len(negative)}')
"
```

- The corpora are NOT included in the repository (file size; reproducibility)
